# AstroAI Bot User Guide - WhatsApp & Telegram

## Quick Start for Users

### Option 1: Chat via WhatsApp (Easiest)
1. **Get the WhatsApp Bot Number** from AstroAI website
2. **Message the bot** with your name
3. **Follow the prompts** to enter your birth details:
   - Date of birth (DD/MM/YYYY)
   - Birth time (HH:MM)
   - Birth place (city name)
4. **Bot generates your kundli** automatically
5. **Ask questions** about your astrological profile

### Option 2: Chat via Telegram
1. **Search for @astroai_bot** on Telegram
2. **Click Start** or send `/start`
3. **Follow the same process** as WhatsApp
4. **Chat about your kundli** with rich formatting and inline buttons

### Option 3: Share from Web App
1. **Generate your kundli** on the AstroAI website
2. **Click "Share to Bot"** button on results page
3. **Choose WhatsApp or Telegram**
4. **Enter your phone number or chat ID**
5. **Receive your kundli** and start chatting

## Troubleshooting

### Bot Not Responding
- **Solution**: Wait a moment and try again
- Check your internet connection
- Make sure you're using the correct bot number/username

### Invalid Date Format
- **Error**: "Invalid date format"
- **Solution**: Use DD/MM/YYYY format (e.g., 15/03/1990)
- Don't use other formats like MM/DD/YYYY

### Invalid Time Format
- **Error**: "Invalid time format"
- **Solution**: Use HH:MM in 24-hour format (e.g., 14:30)
- Don't use AM/PM format

### City Not Found
- **Error**: Bot can't find your city
- **Solution**: Try using a larger nearby city
- Use English spelling of city name

### Kundli Generation Failed
- **Error**: "Failed to generate kundli"
- **Solution**: Check your birth details are correct
- Try again in a few moments
- Contact support if problem persists
